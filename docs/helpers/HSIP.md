# HSIP
Computes the Hilbert-Schmidt inner product between the input matrices

## Syntax
``HSIP(A, B)``

## Argument description
- ``A``: matrix of size m x n
- ``B``: matrix of size m x p

## Example
    >> HSIP(eye(2), eye(2))

    ans =

        2
